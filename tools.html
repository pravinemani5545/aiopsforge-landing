01
Client Onboarding Automation
hybrid
A system that takes a new client from intake form submission to a generated proposal and automated follow-up sequence — zero manual steps. Client fills out a Typeform or Tally form, the system enriches their data, generates a custom proposal using Claude, sends it via email, and triggers a follow-up drip if they don't respond within 48 hours.
MVP Features
  • Webhook trigger on form submission
  • Claude generates custom proposal from template + form data
  • Auto-sends proposal PDF via Gmail/Resend
  • 48-hour follow-up if no reply detected
  • Logs every run to Google Sheets with token cost
Why Hybrid
  • n8n handles the webhook trigger, email sending, and scheduling
  • Claude Code builds the proposal generation logic and prompt chains
  • If selling to clients, n8n's visual canvas makes handoff easy
  • Show token cost per proposal: target under $0.15
n8n claude api tally / typeform gmail / resend google sheets
02
AI Content Repurposing Pipeline
hybrid
Feed in one long-form script, blog post, or video transcript. The system outputs an X thread, 3-5 standalone tweets, a YouTube description, a newsletter section, and TikTok/Reels captions — all in your blended voice. You use this every week for your own content, so it pays for itself immediately.
MVP Features
  • Input: paste or upload long-form text
  • Claude extracts key points and generates each format
  • Voice calibration via system prompt with your style rules
  • Outputs to a structured Google Doc or Notion page
  • One-click scheduling via Typefully API for X posts
Content Output Per Run
  • 1 X thread (8-15 tweets)
  • 3-5 standalone tweets
  • 1 YouTube description with timestamps
  • 1 newsletter block (200-300 words)
  • 2-3 short-form captions for TikTok/Reels
claude api n8n typefully api notion / google docs
03
Competitive Research Agent
claude code agent
A scheduled Claude Code agent that monitors competitors — their websites, social accounts, product pages, pricing changes. Runs daily or weekly via Remote Tasks, compiles a structured digest, and delivers it to you via Telegram or Discord. No manual research sessions needed.
MVP Features
  • Define 3-5 competitor URLs and X handles to monitor
  • Agent scrapes public pages and recent posts
  • Claude analyzes changes, new content, pricing shifts
  • Generates structured markdown report
  • Sends digest via Telegram Channel
Architecture
  • Scheduled Remote Task: runs every Monday 8 AM
  • MCP servers: web scraping, Telegram
  • Stores previous reports for diff comparison
  • Flags significant changes vs noise
  • Token budget cap per run to control costs
claude code remote tasks mcp telegram web scraping

05
AI SEO Content System
hybrid
End-to-end content generation pipeline: keyword research → content brief → article draft → internal linking suggestions → CMS publishing. Document the real cost per article and build time. This is the system behind an AI SEO agency — build it, run it for yourself, then teach it.
MVP Features
  • Input: target keyword or topic cluster
  • Agent researches SERP, analyzes top-ranking content
  • Generates content brief with outline and target length
  • Drafts full article with Claude (with source citations)
  • Suggests internal links from existing content
Production Metrics to Track
  • Token cost per article (target: under $0.50)
  • Time from keyword to published draft
  • Human edit time required post-generation
  • SEO score before vs after optimization pass
  • Indexing and ranking performance over 30/60/90 days

06
Lead Gen & Cold Outreach System
hybrid
Finds prospects from directories and public sources, enriches their data with company info and pain points, generates hyper-personalized cold emails with Claude, and queues them for sending with follow-up sequences. Saraev teaches this as a $1,500-$10,000 sellable automation.
MVP Features
  • Scrape leads from Google Maps, Apollo, or LinkedIn
  • Enrich with company size, industry, tech stack
  • Claude writes personalized first line + email body
  • 3-email follow-up sequence auto-generated
  • Sends via Instantly, Smartlead, or custom SMTP
Edge Cases to Handle
  • Rate limiting on scraping sources
  • Email validation before sending (bounce prevention)
  • Duplicate detection across runs
  • Unsubscribe handling and CAN-SPAM compliance
  • Cost tracking: token spend per lead generated
n8n claude api apollo / apify instantly / smartlead google sheets
07
Workflow Monitoring & Alerting
claude code agent
A meta-system that watches all your other automations. Checks health, monitors token cost spikes, tracks success/failure rates, and alerts you via Telegram when something needs attention. This is the production reliability content that separates you from every demo-only creator.
MVP Features
  • Polls n8n execution logs via API every hour
  • Tracks success rate per workflow over 24h/7d/30d
  • Monitors API spend and flags cost anomalies
  • Sends Telegram alert on failure or threshold breach
  • Weekly summary report: all systems health status
Alert Rules
  • Workflow failure rate exceeds 10% in 24 hours
  • Token spend exceeds 2x daily average
  • Any workflow hasn't run when expected (missed schedule)
  • API response time exceeds 10s (latency degradation)
  • Configurable thresholds per workflow

09
AI Meeting Summarizer
claude code agent
Drop in a meeting recording or transcript. The agent extracts action items, assigns owners, estimates time, and pushes tasks to your project management tool. Cover the failure modes — bad audio, cross-talk, ambiguous commitments — that real production systems need to handle.
MVP Features
  • Input: audio file, transcript, or meeting notes paste
  • Whisper API for audio to text if needed
  • Claude extracts: decisions, action items, owners, deadlines
  • Structured output: JSON or markdown summary
  • Push tasks to Linear, Notion, or Todoist via API
Edge Cases to Handle
  • Ambiguous ownership ("someone should...")
  • No explicit deadlines mentioned — infer or flag
  • Multiple topics in one meeting — section by topic
  • Poor audio quality — confidence scoring on transcript
  • Long meetings (60min+) — chunking for context limits
claude code whisper api linear / notion api telegram mcp

Build-in-Public Journal System

11
SMS AI Task Manager
claude code agent
A to-do list with an AI agent behind it, connected to your phone number. Text it tasks, it organizes them by priority and context. Ask it "what's next?" and it knows your schedule, your current projects, and what's overdue. Persistent memory across every conversation. You carry your entire task system in your text messages.
MVP Features
  • SMS interface via Twilio — text to add/complete/query tasks
  • Natural language: "remind me to ship the video tomorrow 9am"
  • Agent categorizes by project, priority, and deadline
  • Daily morning text: today's tasks ranked by priority
  • Persistent memory: remembers context across all messages
Advanced Features
  • "What's overdue?" — scans all tasks and reports
  • "What did I do this week?" — generates summary
  • Snooze via text: "push the Beehiiv setup to Friday"
  • Project awareness: links tasks to active projects
  • Web dashboard companion for full task view
