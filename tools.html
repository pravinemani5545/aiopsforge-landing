AI Meeting Summarizer
claude code agent coming soon
Drop in a meeting recording or transcript. The agent extracts action items, assigns owners, estimates time, and pushes tasks to your project management tool. Cover the failure modes — bad audio, cross-talk, ambiguous commitments — that real production systems need to handle.
MVP Features
  • Input: audio file, transcript, or meeting notes paste
  • Whisper API for audio to text if needed
  • Claude extracts: decisions, action items, owners, deadlines
  • Structured output: JSON or markdown summary
  • Push tasks to Linear, Notion, or Todoist via API
Edge Cases to Handle
  • Ambiguous ownership ("someone should...")
  • No explicit deadlines mentioned — infer or flag
  • Multiple topics in one meeting — section by topic
  • Poor audio quality — confidence scoring on transcript
  • Long meetings (60min+) — chunking for context limits
