Most "AI agent watches X" tutorials skip the hard parts: rate limits, deduping, prompt cost, where to put state. This build solves all four — MCP server polls every 6 min via Vercel cron, dedupes via Vercel KV, classifies each mention with Claude Sonnet 4.6 (cost: 0.4¢/mention), drops urgent ones into a Slack DM. Total daily cost at 100 mentions: $0.14.
